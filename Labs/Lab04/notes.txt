creat:
Similar to open; if file doesn't exist, create new, then open.

open:
Returns file descriptor of the target file, which monitors interaction with target file

read:
Read from file descriptor

write:
Write to file descriptor

close:
Close a file descriptor

fopen:
Opens file at path and associate a stream with it

fgets:
Reads at most `size-1` characters and put into buffer.

fprintf:
Writes output to given output stream

fseek:
Move file position indicator to `whence+offset` position

fclose:
Close stream
